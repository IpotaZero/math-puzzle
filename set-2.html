§2 差集合


'A,B,C,X:集合'

2.1 ☆☆

'A \subset B'
'x \notin B'

Show: 'x \notin A'

証明 [Claim: 'x \notin A'] 'x \in A'と仮定する 'x \in A \subset B'だから 'x \in B' これは矛盾 よって'x \notin A'

2.2

'x \notin A \cup B'

Show: 'x \notin A'

証明 [Claim: 'x \notin A'] 'x \notin A \cup B'だから '\not (x \in A \cup B)' '\not (x \in A \vee x \in B)' 'x \notin A \wedge x \notin B' とくに 'x \notin A'

2.3 ☆☆☆

'A \subset B'

(1)

Show: 'A \setminus C \subset B \setminus C'

証明 [Claim: 'A \setminus C \subset B \setminus C' i.e. '\forall x \in A \setminus C,x \in B \setminus C'] 'x \in A \setminus C'をとる ['x \in B \setminus C' i.e. 'x \in B \wedge x \notin C'] ['x \in B'] 'x \in A \setminus C' だから 'x \in A \wedge x \notin C' とくに 'x \in A \subset B'
よって 'x \in B'
['x \notin C'] 'x \in A \setminus C'だから 'x \in A \wedge x \notin C' とくに 'x \notin C'

(2)

Show: 'C \setminus B \subset C \setminus A'

証明 [Claim: 'C \setminus B \subset C \setminus A' i.e. '\forall x \in C \setminus B,x \in C \setminus A'] 'x \in C \setminus B'をとる ['x \in C \setminus A' i.e. 'x \in C \wedge x \notin A'] ['x \in C'] 'x \in C \setminus B'だから 'x \in C \wedge x \notin B' とくに 'x \in C'
['x \notin A'] 'x \in C \setminus B'だから 'x \in C \wedge x \notin B' とくに 'x \notin B \supset A'
よって 'x \notin A'

2.4 ☆☆☆☆☆

'A,B \subset X'
(1) 'A \subset B'
(2) 'X \setminus B \subset X \setminus A'

Show: (1) '\Leftrightarrow' (2)

証明 [Claim: (1) '\Leftrightarrow' (2)] ['\Rightarrow'] (1)とする [(2): 'X \setminus B \subset X \setminus A' i.e. '\forall x \in X \setminus B,x \in X \setminus A'] 'x \in X \setminus B' をとる
['x \in X \setminus A' i.e. 'x \in X \wedge x \notin A'] ['x \in X'] 'x \in X \setminus B' だから 'x \in X'
['x \notin A'] 'x \in X \setminus B' だから 'x \notin B \supset A'
よって 'x \notin A'
['\Leftarrow'] (2)とする
[(1): 'A \subset B' i.e. '\forall x \in A, x \in B'] 'x \in A' をとる
['x \in B' i.e. 'x \in X \setminus (X \setminus B)' i.e. 'x \in X \wedge x \notin X \setminus B'] ['x \in X'] 'x \in A \subset X' だから 'x \in X'
['x \notin X \setminus B'] 'x \in A' だから 'x \in X \setminus (X \setminus A)' つまり 'x \in X \wedge x \notin X \setminus A' とくに 'x \notin X \setminus A \supset X \setminus B' よって 'x \notin X \setminus B'

2.5(ド・モドルガンの法則) ☆☆☆☆☆

'A,B \subset X'

(1)
Show: 'X \setminus (A \cup B) = (X \setminus A) \cap (X \setminus B)'

証明 [Claim: 'X \setminus (A \cup B) = (X \setminus A) \cap (X \setminus B)'] ['\subset : \forall x \in X \setminus (A \cup B), x \in (X \setminus A) \cap (X \setminus B)'] 'x \in X \setminus (A \cup B)'をとる
['x \in (X \setminus A) \cap (X \setminus B)'] ['x \in X \setminus A' i.e. 'x \in X \wedge x \notin A'] ['x \in X'] 'x \in X \setminus (A \cup B)'だから 'x \in X'
['x \notin A'] 'x \in X \setminus (A \cup B)'だから 'x \notin A \cup B'
よって 'x \notin A'
['x \in X \setminus B'] ['x \in X'] 上と同様
['x \notin B'] 'x \in X \setminus (A \cup B)'だから 'x \notin A \cup B'
よって 'x \notin B'
['\supset : \forall x \in (X \setminus A) \cap (X \setminus B), x \in X \setminus (A \cup B)'] 'x \in (X \setminus A) \cap (X \setminus B)'をとる
['x \in X \setminus (A \cup B)' i.e. 'x \in X \wedge x \notin A \cup B'] ['x \in X'] 'x \in (X \setminus A) \cap (X \setminus B)'だから 'x \in X \setminus A'
よって 'x \in X'
['x \notin A \cup B'] 'x \in (X \setminus A) \cap (X \setminus B)'だから 'x \notin A \wedge x \notin B'
よって '\not (x \in A \vee x \in B)' '\not (x \in A \cup B)' 'x \notin A \cup B'

(2)
Show: 'X \setminus (A \cap B) = (X \setminus A) \cup (X \setminus B)'

証明 [Claim: 'X \setminus (A \cap B) = (X \setminus A) \cup (X \setminus B)'] ['\subset : \forall x \in X \setminus (A \cap B), x \in (X \setminus A) \cup (X \setminus B)'] 'x \in X \setminus (A \cap B)'をとる
['x \in (X \setminus A) \cup (X \setminus B)'] 'x \in X \setminus (A \cap B)'だから 'x \in X \wedge x \notin A \cap B' とくに 'x \in X' 'x \notin A \vee x \notin B'
'x \notin A'のとき 'x \in X \setminus A'
'x \notin B'のとき 'x \in X \setminus B'
['\supset : \forall x \in (X \setminus A) \cup (X \setminus B), x \in X \setminus (A \cap B)'] 'x \in (X \setminus A) \cup (X \setminus B)'をとる
['x \in X \setminus (A \cap B)'] ['x \in X'] 'x \in (X \setminus A) \cup (X \setminus B)'だから 'x \in X \setminus A \vee x \in X \setminus B'
'x \in X \setminus A'のとき 'x \in X'
'x \in X \setminus B'のとき 'x \in X'
['x \notin A \cap B' i.e. '\not (x \in A \wedge x \in B)' i.e. 'x \notin A \vee x \notin B'] 上と同様に 'x \in X \setminus A \vee x \in X \setminus B'
'x \in X \setminus A'のとき 'x \notin A'
'x \in X \setminus B'のとき 'x \notin B'
